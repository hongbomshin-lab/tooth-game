# 🦷 치과 발치 리듬 게임 (Dentistry Rhythm Game)

화살표 키를 이용해 박자에 맞춰 치아를 발치하는 역동적인 리듬 게임입니다.

## 🎮 게임 특징
- **2가지 난이도**: 쉬움(3방향), 어려움(4방향) 모드 지원
- **실시간 비주얼**: 발치 게이지에 따라 치아가 실제로 뽑히는 애니메이션
- **드라마틱한 연출**: 성공 시 화려한 폭죽(Confetti)과 화면 진동 효과
- **반응형 디자인**: 세련된 다크 모드 테마와 부드러운 UI 

## 🕹️ 플레이 방법
1. [게임 플레이 링크](https://[사용자아이디].github.io/치과게임/) (배포 후 링크를 수정하세요)
2. 시작 화면에서 **난이도**를 선택합니다.
3. 내려오는 화살표에 맞춰 **방향키(↑, ←, ↓, →)**를 누릅니다.
4. 상단의 발치 게이지가 100%가 될 때까지 점수를 획득하세요!

## 🛠️ 기술 스택
- HTML5
- Vanilla CSS (Procedural Visuals)
- Pure JavaScript (Game Engine)

## 📁 파일 구조
- `index.html`: 게임 구조 및 UI
- `style.css`: 게임 디자인 및 애니메이션 효과
- `game.js`: 리듬 엔진 및 게임 로직
